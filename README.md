# Discord Summarizer Bot
The Discord Summarizer bot "StudyBuddy" is a powerful tool designed to make studying, project work, and team collaboration easier and more efficient.
With the /summarize slash command, users can quickly condense long text, meeting notes, or research material into concise summaries, saving time and helping retain key information.

## Whether you're:
- Reviewing lecture notes or study material
- Summarizing project documentation and timelines
- Keeping track of team meetings or brainstorming sessions
...this bot provides instant, AI-generated sumarries right in your Discord server.

## Features
- '/summarize' command to summarize text
- Real-time summarization
- Safe API key handling
- Error-handling
- Cross-Platform
- Easy Setup
